# ai-article-auto-pipeline-exe
AI驱动一体化学术/科普文章生成流水线工具

## 📌 项目开发目标
本程序为AI驱动一体化文章生成流水线，自动化完成完整学术或科普文章产出，输出5000字以上图文搭配文稿，包含不少于40篇参考文献，支持投稿材料打包、自媒体文案生成。
> 课堂Demo说明：移除账号自动登录、数据库爬虫等高风险自动化功能，仅演示调度框架。

## ⚙️ 四大并行子任务调度
1. 写文章模块：选题、参考文献清单生成、文稿初稿撰写
2. 绘图模块：AI绘图提示词生成、投稿压缩包打包
3. 作者期刊信息整理模块：输出期刊投稿模板文档
4. 自媒体文案Demo模块：生成发布文案（无自动登录）

## 🛠️ 核心执行流程
1. 调度程序并发启动四大子任务
2. 各任务独立执行，输出本地资源文件
3. 整合全部资源生成完整投稿文稿交付包
